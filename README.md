# deep-learning-from-scratch
2022년 동계 방학 KERT 프로젝트/스터디: 딥러닝 스터디. Numpy를 이용해 퍼셉트론, 신경망, 오차 역전파 등을 밑바닥부터 구현해봄으로써 딥러닝의 기본 개념에 대해 학습하고, 이해한다.

## 스터디 멤버
- **신찬규 (스터디 장)**
- 윤경록

## 도서
- 사이토 고키, 『밑바닥부터 시작하는 딥러닝: 파이썬으로 익히는 딥러닝 이론과 구현』, 한빛미디어
